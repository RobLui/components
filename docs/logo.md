# BD-LOGO

## Logo
### Styling

`.ng2-c-logo`

A container for the logo.

### Usage

```html
<div class="ng2-c-logo">
    <i class="icon-delen"></i>
</div>
```

## Logo App

### Styling

`.ng2-c-logo--app`

### Usage

**Logo App**
```html
<span class="c-logo--app"></span>
```

**Logo App Oyens**
```html
<body class="body--oyens">
    <span class="c-logo--app"></span>
</body>
```

**Logo App Suisse**
```html
<body class="body--delench">
    <span class="c-logo--app"></span>
</body>
```

**Logo App Luxembourg**
```html
<body class="body--delenlu">
    <span class="c-logo--app"></span>
</body>
```

Display the app logo.
